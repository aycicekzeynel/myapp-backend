================================================================
🚀 BACKEND HAZIR - ŞİMDİ NE YAPACAKSIN?
================================================================

Terminal'de şu komutları çalıştır:

cd /Users/zeynelaycicek/projects/myapp-backend

# Script'lere izin ver
chmod +x quick-start.sh test-api.sh

# Sunucuyu başlat (otomatik - önerilen)
./quick-start.sh

# VEYA manuel başlat
npm install
npm run dev

================================================================

✅ BAŞARILI KURULUM GÖSTERGELERİ:

Server çalıştığında şunu görmelisin:
- "MongoDB'ye bağlanıldı: myapp"
- "Server Started Successfully"
- "Host: http://0.0.0.0:5000"

curl http://localhost:5000/health

📱 MOBİL UYGULAMAYA BAĞLA:

1. Backend'i çalıştır
2. Local IP'ni bul:
   ifconfig | grep "inet " | grep -v 127.0.0.1
   
3. myapp-mobile-v4 projesinde API_URL'i güncelle:
   http://YOUR_LOCAL_IP:5000

================================================================

🔧 SORUN YAŞARSAN:

1. MongoDB çalışıyor mu?
   brew services list | grep mongodb
   
2. node_modules yüklü mü?
   npm install
   
3. Port boş mu?
   lsof -ti:5000 | xargs kill -9
